# Güzellik salonu websitesi
Güzellik Merkezi - Randevu Sayfası

Bu proje, bir güzellik merkezi için tasarlanmış, sade, modern ve kullanıcı dostu bir tek sayfalık (Single Page) randevu kayıt arayüzüdür.

## Özellikler

Minimalist Tasarım: Pastel tonlar ve temiz bir arayüz ile profesyonel görünüm.

Mobil Uyumluluk: Responsive yapısı sayesinde telefon, tablet ve masaüstü cihazlarda sorunsuz çalışır.

Hızlı Etkileşim: Form gönderildiğinde sayfa yenilenmeden kullanıcıya başarı mesajı gösterilir.

Tek Dosya: HTML, CSS ve JavaScript tek bir dosya içinde yapılandırılmıştır, kurulum gerektirmez.

## Kullanılan Teknolojiler

HTML: Sayfa yapısı ve içerik iskeleti için.

CSS: Görsel tasarım, renk paleti ve kart düzenleri için.

JavaScript: Form yönetimi ve temel kullanıcı etkileşimi için.

## Kurulum ve Kullanım

Bu depoyu klonlayın veya index.html dosyasını indirin.

Dosyaya çift tıklayarak herhangi bir tarayıcıda (Chrome, Safari, Edge vb.) görüntüleyebilirsiniz.

*Bu proje eğitim amaçlı, başlangıç seviyesinde bir web uygulaması örneği olarak geliştirilmiştir.

 # beauty salon website
This project is a simple, modern, and user-friendly single-page appointment booking interface designed for a beauty center.
## Features
Minimalist Design: A professional look with pastel tones and a clean interface.
Mobile Responsive: Works seamlessly on phones, tablets, and desktop devices.
Instant Interaction: Displays a success message upon form submission without refreshing the page.
Single File Structure: HTML, CSS, and JavaScript are all contained within one file; no complex setup required.
## Technologies Used
HTML: For page structure and content skeleton.
CSS: For visual design, color palette, and card layouts.
JavaScript: For form management and basic user interactions.
## Installation and Usage
Clone this repository or download the index.html file.
Double-click the file to view it in any web browser (Chrome, Safari, Edge, etc.).
*This project was developed as a beginner-level web application example for educational purposes.
